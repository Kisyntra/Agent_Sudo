# Recording the exfiltration demo (GIF / video)

Goal: a ‚Č§60s clip showing (1) the poisoned page, (2) `python examples/exfil_demo/demo.py`,
(3) `agent-sudo verify-audit`. Two reproducible paths are provided. **No code is
changed** ‚ÄĒ these are recording assets only.

Prereq for both: install the package so the CLI/demo run, from the repo root:

```bash
pip install -e .
```

---

## Option A ‚ÄĒ VHS (recommended, fully scripted & deterministic)

[VHS](https://github.com/charmbracelet/vhs) renders a GIF from a script ‚ÄĒ no manual typing, identical every time.

```bash
brew install vhs           # or: go install github.com/charmbracelet/vhs@latest
cd <repo root>
vhs examples/exfil_demo/recording/demo.tape
```

Produces **`exfil-demo.gif`** in the current directory (~25s, 1280√ó760).
The tape types each command, runs it, and pauses so the three verdicts and the
`audit log verified` line are readable.

Tune in `demo.tape` if needed:
- `Set Theme` ‚ÄĒ any VHS theme (`vhs themes` to list).
- `Set FontSize` / `Set Width` / `Set Height` ‚ÄĒ keep width ‚Č• ~1100px so the
  91-col output line isn't wrapped.
- `Sleep` values ‚ÄĒ adjust dwell time per step (currently ~4.5s / 7s / 3s).

## Option B ‚ÄĒ asciinema + agg (terminal cast ‚Üí GIF)

Use the scripted driver when you want the same command pacing without VHS:

```bash
brew install asciinema agg
cd <repo root>
asciinema rec --overwrite -c "bash examples/exfil_demo/recording/record.sh" exfil-demo.cast
agg exfil-demo.cast exfil-demo.gif
```

Or record manually:

```bash
brew install asciinema agg
cd <repo root>
asciinema rec --overwrite exfil-demo.cast   # then type the three commands below; Ctrl-D to stop
agg exfil-demo.cast exfil-demo.gif
```

The three commands to type (pause a few seconds on each so output is readable):

```bash
cat examples/exfil_demo/poisoned_page.txt
python examples/exfil_demo/demo.py
agent-sudo verify-audit examples/exfil_demo/audit.jsonl
```

Any screen recorder (QuickTime, Kap, OBS) works the same way if you prefer a video.

## Notes

- The path the demo prints in its last line is absolute (it reflects your local
  checkout) ‚ÄĒ cosmetic only; the command you actually type in step 3 uses the
  short relative path.
- Suggested home for the rendered file: `assets/demo/exfil-demo.gif`, then it can
  be embedded in `examples/exfil_demo/README.md` and/or the root README.
- `examples/exfil_demo/audit.jsonl` is regenerated by `demo.py` on each run and is
  gitignored, so recording never depends on stale state.
